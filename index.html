What Elyne is

Elyne is a premium personal-growth app designed for reflective users who want a visual, structured way to imagine the life they are building. Instead of text-only journaling or generic vision-boards, Elyne guides the user through nine spheres of life and helps them curate a coherent visual moodboard for each.

The app combines onboarding conversations, archetype selection (a curated set of visual-style references), and image-collection tools. Users can connect their Pinterest account to bring in visual material they already love, and Elyne organises it inside a structured life-design framework.

Our design language is quiet, editorial, and warm — built on the aesthetic intuition of Apple, Notion, and Calm.

Pinterest integration

Elyne connects to Pinterest via the official Pinterest API so that users can pull in their existing pins and boards as starting material for each life sphere. This is an opt-in feature — users authenticate with their Pinterest account through OAuth, and Elyne reads only the public pin and board metadata the user explicitly authorises.

What Elyne accesses on Pinterest

  • Public pin metadata — image URL, caption, board assignment
  • Public board metadata — name, description, cover image
  • Basic user account info — account identifier, authentication only

What Elyne does not do

  • No access to Pinterest passwords — authentication is handled entirely by Pinterest via OAuth
  • No access to private boards or private pins without explicit permission
  • No scraping of Pinterest content outside the official API
  • No selling or sharing user Pinterest data with third-party advertisers
