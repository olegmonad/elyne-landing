================================
ШАГ 3 — СПЕЦИФИКА СФЕРЫ 1
================================

**Главный сюжет:** как человек создаёт, держит и проживает своё состояние.

**Это сцена про:** личную опору, телесную ресурсность, внутреннюю собранность,
живую энергию, качество контакта с собой.

**Это НЕ сцена про:** работу, влияние, проекты, статус. Даже если в ответах
есть «масштаб / влияние / реализация» — здесь они проявляются ТОЛЬКО косвенно,
через качество жизни, не как главный сюжет.

Выбери:

**Тип Сцены** (один):
— private_life → личный качественный фрагмент
— engagement → включённость в настройку состояния
— movement → движение как способ собрать себя
— exploration → исследование нового качества жизни через тело и ритм

**Тип Действия** (один):
— moving / interacting / observing / creating / deciding

**Роль** (один доминирующий self-тип):
— embodied / disciplined / balanced / grounded / recovering / vital / elegant / explorer

Все три должны быть согласованы между собой и с Механикой × Телом из ШАГ 2.

ШАГ 4 — СЦЕНАРИЙ ИЗ БИБЛИОТЕКИ
================================

Сначала выбери 3 кандидата, потом — ОДИН лучший.

**[PRIVATE LIFE]** утренний ритуал · личный момент настройки · спокойное время
высокого качества · после воды/воздуха/тишины/практики · переход в день через
собранность · естественное поддержание себя в красивой повседневности

**[ENGAGEMENT]** телесная практика без фитнес-клише · взаимодействие со средой ·
настройка пространства под свой ритм · дыхание/движение/мягкая нагрузка/растяжение/
вода как живой процесс · собирает себя через действие · действие, через которое
создаётся энергия

**[MOVEMENT]** проход через пространство как настройка · движение вдоль террасы/
сада/двора/берега · вход в ритм через походку · перемещение между частями
пространства как ритуал · спокойное осмысленное движение

**[EXPLORATION]** исследование нового качества через тело · проживание себя в
новом ритме · открытие более здорового/живого/устойчивого состояния · включённость
в поддерживающее пространство · пробует более ресурсный способ быть собой

**Если можно подставить любого человека — невалидно.** Если банально — пересобери.

================================
ШАГ 5 — ПОСТРОЕНИЕ СЦЕНЫ
================================

**5.1 Композиция и план**
Передний / средний / задний / что за кадром. Сцена — момент внутри жизни, не поза.

**5.2 Кадр** (выбери осознанно):
— close → состояние, лицо, микродетали тела, дыхание, руки, кожа
— medium → тело + действие + пространство одновременно
— wide → среда, воздух, способ жизни (тело всё равно читается как часть процесса)

Связка с типом тела/механики:
body/energy/embodied → чаще close/medium · space/recovery → чаще medium/wide.

**Узнаваемость лица:**
— на close/medium лицо чёткое и узнаваемое; состояние через позу/плечи/руки/походку
— на wide лицо не главное, не детализировать; читаются тело и движение

**5.3 Атрибутика и среда**
Принцип: уровень жизни + механика + тип тела → пространство, материалы, свет,
воздух, предметы.

Допустимо: пространство высокого качества, вода/тень/свет/терраса/сад/дерево/камень/
текстиль/архитектура; телесные объекты без клише (коврик, полотенце, плед, скамья,
ступени, бассейн, душ, природная дорожка); lived-in details (следы жизни,
несовершенства).

Запрещено: random luxury, wellness ради красивости, проектные/рабочие объекты,
статусные символы без связи с телом, предметы которые можно подставить любому.

**5.4 Внешний вид и одежда**
Внешний вид — продолжение состояния и телесности, не отдельный шаблонный стиль.

Одежда рождается из связки тип_тела × роль × механика:
— grounded → спокойная, тактильная, устойчивая
— light → лёгкая, дышащая, свободная
— disciplined → собранная, чистая, функциональная
— elegant → пластичная, красивая в движении, минималистичная
— recovery → мягкая, живая, комфортная, не безликая

`style_profile` пользователя используется как **код-фильтр**: из выбранных юзером
3-5 стилей выбери тот, который ближе всего к выведенному «визуальному коду
состояния». Если ни один не подходит идеально — берём наименее доминирующий.

================================
ШАГ 6 — ЛИЦО И ТЕЛО (РЕАЛИЗМ + УЗНАВАЕМОСТЬ)
================================

**Лицо:** узнаваемое, без искажений, без глянца, без потери индивидуальности.
**Тело:** естественная анатомия и пропорции, живая пластика, правдоподобная поза.

**Естественное улучшение допустимо:**
— общий вид аккуратнее, тон кожи живее, лицо ясное
— осанка более собранная (без искусственности)
— волосы аккуратнее
— смягчить следы усталости

**Запрещено:** пластик, омоложение, превращение в модель, потеря индивидуальности,
нереалистично идеальное тело, фитнес-реклама.

================================
ШАГ 7 — ПЕРЕВОД СОСТОЯНИЯ В ВИЗУАЛ
================================

Определи доминирующее состояние сцены:
устойчивость · восстановление · мягкая сила · собранность · свобода внутри себя ·
живая энергия · телесная ясность · спокойный ресурс · пластичное присутствие

→ Переведи в визуальные параметры:

**СВЕТ:**
— устойчивость → мягкий рассеянный, опорный
— собранность → чистый направленный
— восстановление → тёплый мягкий
— живая энергия → ясный оживляющий, не агрессивный
— свобода → воздух, открытость, мягкая яркость
— пластичность → свет, красиво читающий тело в движении

**ЦВЕТ И АТМОСФЕРА:** нейтральные, живые, тактильные оттенки. Без кислотности,
без декоративной «духовности».

**ТЕЛО И ПОЗА:** естественная осанка, дыхание, включённость, без напряжения ради
картинки. Тело — живой носитель состояния.

**ДВИЖЕНИЕ:** плавное, живое, осмысленное. Часть ритма саморегуляции, не поза
ради эстетики.

ШАГ 8 — АНТИ-ШАБЛОН
================================

**Запрещённые действия:** держит чертежи/документы; обсуждает работу; деловая
функция; влияние через жесты; кофе/чай/кружка; телефон; стоит у зеркала; «сидит
красиво и осознанно»; гуляет без связи с состоянием; спортзал как универсальный
символ; йога-как-дефолт; спа-декорации без смысла.

**Запрещённые типы сцен:** wellness lifestyle, реклама спа, сток про self-care,
красивая но пустая lifestyle-сцена, «человек который просто хорошо выглядит в
хорошей среде».

**Запрещённые композиции (вариативность):**
— дефолт «человек в центре + другие вокруг + объясняет» → запрещено
— меняй композицию: центр / сбоку / частично / вне фокуса / часть среды / без человека / POV
— меняй роль: лидер не обязателен; равный, наблюдающий, присутствующий, отсутствующий — допустимы
— меняй действие: активное / мягкое / наблюдение / пауза / статичность / отсутствие действия

**Запрещены эмоции словами** (happy, smiling, peaceful, mindful, joyful) и
self-help термины (mindfulness, gratitude, manifestation, journey, balance).

**Тест уникальности:** можно ли сцену перенести в другую сферу без изменений?
Если да — слабая, пересобери.

**Сцена должна содержать минимум 2 из:**
— ощущение пойманного момента
— жизнь за пределами кадра
— реальный телесный процесс
— взаимодействие со средой
— читаемая механика состояния
— лёгкая естественная неидеальность
