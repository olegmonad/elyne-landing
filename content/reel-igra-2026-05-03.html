Идея визуального языка

Не копирую Алфёрева (там тёмный AI-глитч, эзотерика-near). Олегу нужно — premium, как Apple Calm-реклама + сквозная метафора шахматы (она у него в тексте: «партия», «ход», «роль»). Цветовой кодекс: charcoal-sepia + один янтарный акцент. Без лиц, без manifestation-визуала.

Сквозные правила всех шотов

  • 9:16, 1080×1920, 24-30 fps
  • Без лиц, без читаемых людей — только силуэты, руки, фигуры на дистанции
  • Палитра: sepia-charcoal monochrome + одна точка тёплого янтаря (свет, акцент)
  • Свет: мягкий sidelight, golden-hour либо studio-amber
  • Камера: медленные dolly/track shots, никаких рывков. Slo-mo на ключевых моментах
  • Без текста в кадре (текст добавим потом через ffmpeg burn-in субтитры)
  • Длина клипа: 5-10 сек, склейка прямым cut’ом или мягким crossfade на 0.5 сек
  • Запрещено: глитч, неоновые цвета (кроме одного янтаря), частицы-эзотерика, чакры, гипноз-спирали, чары

Стек реализации

  1. Voiceover — ты записываешь голосом (либо я через edge-tts если хочешь машинный — но твой голос правильнее)
  2. Whisper — транскрипт + word-level timestamps для синхронизации субтитров
  3. Video-генерация — каждый промт выше → один клип 5-10 сек: - Sora 2 (через ChatGPT Plus / API): premium качество, ~$1-3 за клип - Google Veo 3 (через Gemini): сопоставимо - FAL FLUX video (через MCP): ~$0.5-1 за клип, но качество ниже Sora - Runway Gen-4: $1-2 за клип
  4. Монтаж ffmpeg: - склейка 8 клипов по таймингу - voiceover как основная audio track - ambient drone подложкой -25 dB (взять любой free royalty) - burn-in субтитров через drawtext по word-timestamps из whisper - финал 9:16, h.264 / mp4
