Что Олег увидел (расшифровка)

1. «Формат не для рилса в Instagram, расширен»

Технически 1080×1920 правильный. Но визуально кадр читается как горизонтальный — в шотах 1, 3, 5 композиция исходных клипов изначально landscape (потолки/коридоры/панорама), и при scale+pad в 9:16 остаётся горизонтальная логика. Глаз видит «вертикально обрезанное горизонтальное видео».

Шот 1: массивный «потолок + люстры» в верхней половине, король внизу справа теряется. Не king-shot.

2. «Шрифт ужасный»

DejaVu Sans 14pt с тенью — системный default Linux, выглядит как Aegisub 2008 / fanmade YouTube середины 2010-х. Premium-стандарт сейчас:

  • Inter / SF Pro / Helvetica Now / Söhne / TT Norms Pro
  • Weight 300-400 (Light/Regular), не Bold
  • Letter-spacing +20 до +40
  • Размер 52-64 px (а не 14pt)
  • Тень — очень мягкая (blur 8, alpha 30%), не drop-shadow
  • Цвет — off-white #F5F2EB (тёплый бежевый), не чисто белый

3. «Слова должны появляться и пропадать»

SRT мёртвый. Премиум — ASS subtitles с fade на каждую фразу:

Style: Default,Inter,58,&H00EBF2F5,&H00000000,&H80000000,0,0,1,0,0,5,80,80,200
Dialogue: 0,0:00:01.00,0:00:05.00,Default,,0,0,0,,{\fad(400,400)\blur0.6}Тебе никто не говорил, что это игра.
  • \fad(400,400) — fade in 400ms, out 400ms
  • \blur0.6 — gaussian blur по краям, премиум-софт
  • Bottom-third (MarginV=380), не центр-середина

Между фразами 30-50ms полной пустоты — глаз отдыхает. Субтитр приходит на 150ms раньше voice, уходит на 200ms позже.

Для отдельных слов («Надо.» «Правильно.» «А что скажут.») — появление с интервалом 800ms, не одновременно. Тяжесть нарастает.

4. «Качество колхозное» — конкретные дефекты

  1. Шрифт (главный)
  2. Субтитры по центру, не bottom-third
  3. Нет fade — текст «вываливается»
  4. Нет общего LUT поверх — каждый клип со своим цветом, нет единого фильма
  5. Нет film grain поверх склейки
  6. Voice не сидит в пространстве (см. ниже)
  7. Нет safe-area padding (нижние 250px перекрываются UI Instagram)

5. «Голос слишком громкий» — audio mix

Voice -16 LUFS + ambient -25dB = разрыв 12-15dB, voice доминирует. Премиум-стандарт:

Слой Уровень
Voice -18 to -20 LUFS short-term
Ambient drone -24 to -22 LUFS
Foley accents пик -28 to -24 dBFS
Integrated loudness -14 LUFS (стандарт IG)
True peak ≤ -1.0 dBTP

Разрыв voice/ambient — 6-8 dB, не 12. Дополнительно: - High-pass voice 80 Hz - Sidechain ducking ambient -3dB при voice - На паузах ambient возвращается — даёт «дыхание» mix’у

Top-3 правки приоритетом для v2

  1. Шрифт + анимация (Inter Light 56px, fade 350/300ms, blur 0.4, bottom-third) — +60% воспринимаемого качества
  2. Audio mix (voice -19 LUFS, ambient -22dB с ducking, integrated -14) — голос перестанет торчать
  3. Единый LUT + film grain — связывает 8 разных AI-клипов в один фильм

Конкретные параметры для ffmpeg/ASS

ASS subtitle style

Font: Inter Light (300), size 56, color #EBE7DC
Letter-spacing: +30
Line-height: 1.3
Outline: 0
Shadow: blur 12 alpha 25% offset 0,0
Position: bottom-third (MarginV=380)
Max width 880px, max 2 строки

Audio chain

# voice
highpass=f=80, equalizer=f=200:g=-2, equalizer=f=3000:g=2,
acompressor=threshold=-18dB:ratio=3, loudnorm=I=-19:TP=-1.5

# ambient with ducking
volume=-22dB, sidechaincompress=threshold=-30dB:ratio=4

# final integrated -14
amix + loudnorm=I=-14:TP=-1.0

Color grade LUT (поверх всех клипов)

curves=preset=increase_contrast,
colorbalance=rs=0.05:gs=-0.02:bs=-0.08:rm=0.03:bm=-0.05,
eq=saturation=0.85:contrast=1.08:gamma=0.95,
noise=alls=8:allf=t

Композиция шотов 1, 3, 5

Перегенерировать с явным «9:16 vertical composition, NOT horizontal cropped», или crop по правилу третей где главный объект в правом-верхнем третях.

Safe area для Instagram

  • Subtitle MarginV ≥ 380px от низа
  • Top safe ≥ 220px (header IG)
  • Текст в средней трети по горизонтали, padding 60px от краёв
