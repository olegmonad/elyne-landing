  1. onboarding_answers + profile_summary — смысловой вектор
  2. sphere_id → field_scene prompt (тема сцены)
  3. Key Answer Locks — жёсткие ограничения
  4. scene_system ID=11 — финальные проверки (role-performance, cliché, style overpowering)
  5. clothing_style description — визуальное направление (не архетип)
  6. image_instructions ID=12 — rendering правила
  7. image_suffix — суффикс к FAL (есть у ID=1 и ID=2)

Стиль одежды больше не определяет: профессию, место, lifestyle archetype, сценарий.
