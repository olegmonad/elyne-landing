Style Profiles · 10 стилей · ♀ / ♂

Формат жёсткий, по шаблону. Materials / Color Logic / Social Signal / Best Scene Fit / Avoid — общие для стиля. Base Wardrobe разделён по гендеру, потому что там реальная разница в крое и силуэте.

Style Profile: Old Money

Style Name: Old Money

Visual Code: сдержанный, статусный, без логотипов. Тихая роскошь, наследуемый вкус.

Base Wardrobe ♀: - Верх: блузка из шёлка, поло, тонкий кашемир-свитер, белая рубашка, жилет. - Низ: юбка-миди карандаш или плиссе, прямые шерстяные брюки, светлые джинсы. - Платья: платье-футляр миди, шёлковое slip-платье в спокойных оттенках. - Обувь: балетки, лоферы (penny / horsebit), kitten heel-лодочки, мягкие мокасины. - Верхняя одежда: тренч, шерстяное пальто, кашемировое пальто длинного силуэта. - Аксессуары: жемчуг, тонкие золотые цепочки, шёлковый шарф, премиум-сумка одной формы (top-handle).

Base Wardrobe ♂: - Верх: белая или сорочечная рубашка, поло, тонкий кашемир-свитер, тонкий джемпер, твидовый блейзер. - Низ: чиносы из плотного хлопка, прямые шерстяные брюки, светлые джинсы. - Обувь: лоферы (penny / horsebit), мокасины, balmoral-туфли, derby. - Верхняя одежда: тренч, шерстяное пальто, кашемировое пальто. - Аксессуары: часы простой формы, кожаный ремень, шёлковый платок в нагрудный карман, премиум-портфель.

Materials: кашемир, твид, шёлк, тонкая шерсть, лён, премиум-хлопок.

Color Logic: тёплые нейтральные — beige, cream, navy, camel, soft brown, off-white. Один акцент на образ.

Social Signal: статус без демонстрации, ухоженность, наследуемый вкус, спокойная сила.

Best Scene Fit: Достаток · Образ-Я · Отношения (ресторан, ужин) · Социальность (избранный круг).

Avoid: блестящие логотипы, trend-асимметрия, urban street, технологичные ткани, brand-overload.

Style Profile: Business Style

Style Name: Business Style

Visual Code: структурный, властный, современная деловая эстетика. Читаемая компетентность.

Base Wardrobe ♀: - Верх: блузка, шёлковая рубашка, тонкий джемпер, водолазка под пиджак. - Низ: юбка-карандаш, прямые брюки структурного кроя, шерстяные брюки от костюма. - Платья: платье-футляр под пиджак, миди-платье плотного кроя. - Обувь: лодочки на низком каблуке (5–7 см), оксфорды-лодочки, лоферы. - Верхняя одежда: пиджак slim, костюм-двойка, пальто-кокон. - Аксессуары: тонкий ремень, минимальные украшения, кожаная сумка-тоут или briefcase, часы.

Base Wardrobe ♂: - Верх: рубашка с галстуком или без, тонкий джемпер, водолазка под пиджак. - Низ: брюки от костюма, классические шерстяные брюки. - Обувь: derby, monk-strap, лоферы. - Верхняя одежда: костюм-двойка/тройка slim или чуть-оверсайз, пальто-кокон, тренч. - Аксессуары: часы, ремень, портфель, шёлковый галстук или платок в нагрудный карман.

Materials: качественная шерсть, хлопок-поплин, шёлк (галстук / блузка), плотный трикотаж, кожа.

Color Logic: графит, чёрный, белый, navy. Бордо или глубокий зелёный как редкий акцент.

Social Signal: компетентность, сила, контроль, ясность.

Best Scene Fit: Проявленность · Достаток · Образование (лекция, конференция).

Avoid: широкий галстук 90-х, canvas-рюкзак, мятая рубашка, pastel-оттенки, двубортный «корпоратив-2000».

Style Profile: Classic

Style Name: Classic

Visual Code: вечная база, без субкультурного шума. То, что не устаревает.

Base Wardrobe ♀: - Верх: белая рубашка, чёрный тонкий джемпер, водолазка. - Низ: бежевые брюки, прямые джинсы, юбка миди A-line. - Платья: чёрное платье-футляр, миди-платье в полоску или клетку. - Обувь: балетки, лоферы, лодочки на низком каблуке. - Верхняя одежда: тренч, шерстяное пальто. - Аксессуары: тонкий пояс, часы, лаконичная кожаная сумка, шёлковый платок.

Base Wardrobe ♂: - Верх: белая рубашка, тонкий чёрный джемпер, кашемировый свитер. - Низ: бежевые чиносы, прямые джинсы, шерстяные брюки. - Обувь: лоферы, derby, кожаные туфли на низком каблуке. - Верхняя одежда: тренч, шерстяное пальто. - Аксессуары: ремень, часы, лаконичная сумка / портфель.

Materials: хлопок, шерсть, кожа, плотный трикотаж, лён (межсезонье).

Color Logic: чёрный, белый, беж, navy в мягких сочетаниях. Принт — клетка, полоска, пейсли (редко).

Social Signal: вкус без претензии, надёжность, мягкая уверенность.

Best Scene Fit: Образ-Я · Семья (нейтральная база) · Социальность · Образование.

Avoid: subculture-маркеры, яркая мода, избыточный декор, total-trend-look.

Style Profile: Casual (minimal-luxe)

Style Name: Casual (minimal-luxe)

Visual Code: повседневный, но материал говорит за себя. Не выглядит как усилие.

Base Wardrobe ♀: - Верх: плотная белая футболка, тонкий кашемир-свитер, рубашка-оверсайз, лёгкая блузка. - Низ: прямые джинсы, льняные брюки, миди-юбка прямого кроя. - Платья: льняное миди, шёлковое slip-платье в нейтральном цвете. - Обувь: lifestyle-кроссовки (не спорт), лоферы, мягкие мокасины, замшевые мюли. - Верхняя одежда: кардиган длинного силуэта, лёгкая куртка-shacket. - Аксессуары: одна сумка-тоут, часы.

Base Wardrobe ♂: - Верх: плотная белая футболка, тонкий кашемир-свитер, рубашка-оверсайз. - Низ: прямые джинсы, льняные / хлопковые брюки, чиносы. - Обувь: lifestyle-кроссовки (не спорт), лоферы, замшевые desert-boots. - Верхняя одежда: кардиган, лёгкая куртка-shacket. - Аксессуары: одна сумка, часы.

Materials: кашемир, плотный хлопок, лён, замша, плотный трикотаж.

Color Logic: off-white, soft grey, warm beige, muted navy.

Social Signal: вкус, лёгкость, естественность, благополучие.

Best Scene Fit: Семья · Образ-Я · Творчество (home-studio) · Образование (коворкинг).

Avoid: спортзал-look, brand-overload, pastel-шум, fast-fashion-силуэт.

Style Profile: Streetwear (premium)

Style Name: Streetwear (premium)

Visual Code: уличный fashion-forward, не массовый hip-hop. Точный крой при оверсайзе.

Base Wardrobe ♀: - Верх: премиум-худи оверсайз, графический long-sleeve, оверсайз-рубашка, премиум кроп-топ. - Низ: премиум-деним baggy / wide-leg, прямые широкие брюки, оверсайз-карго. - Обувь: массивные lifestyle-кроссовки (не беговые), platform-кеды. - Верхняя одежда: оверсайз-куртка, бомбер, анорак. - Аксессуары: crossbody, beanie, кепка, plaque-серьги.

Base Wardrobe ♂: - Верх: премиум-худи, графический long-sleeve, оверсайз-рубашка. - Низ: прямые / широкие брюки, премиум-деним, карго. - Обувь: массивные lifestyle-кроссовки (не беговые). - Верхняя одежда: анорак, оверсайз-куртка, бомбер. - Аксессуары: crossbody, beanie, кепка.

Materials: ripstop, нейлон, плотный premium-хлопок, деним, технический трикотаж.

Color Logic: графит, хаки, приглушённый вишнёвый, выцветший индиго. Бренд-акцент — редко и точечно.

Social Signal: креативность, fashion-литература, городская независимость.

Best Scene Fit: Творчество · Социальность · Путешествия (city break) · Проявленность (креативные индустрии).

Avoid: total-look бренда, sportswear, fast-fashion подделки, sneakerhead-нагромождение.

Style Profile: Sport Casual (function + luxury)

Style Name: Sport Casual (function + luxury)

Visual Code: спорт-повседневный с одним luxury-якорем (часы / куртка / сумка).

Base Wardrobe ♀: - Верх: худи slim, технический long-sleeve, sport-bra premium, тонкий бомбер. - Низ: технические leggings, jogger-крой, lifestyle-шорты. - Обувь: lifestyle-кроссовки одной премиум-модели. - Верхняя одежда: лёгкая shell-куртка, пуховик slim-fit. - Аксессуары: часы или сумка как luxury-якорь, спортивная кепка.

Base Wardrobe ♂: - Верх: худи, технический long-sleeve, тонкий бомбер. - Низ: технические брюки, jogger-крой. - Обувь: lifestyle-кроссовки одной премиум-модели. - Верхняя одежда: лёгкая shell-куртка, пуховик slim-fit. - Аксессуары: часы или сумка как luxury-якорь.

Materials: технический хлопок, флис, мембраны (Gore-Tex / shell), плотный трикотаж.

Color Logic: чёрный, графит, slate-blue, олива. Белый как контраст.

Social Signal: дисциплина, ритм, тихий статус.

Best Scene Fit: Образ-Я (утро, тренировка) · Путешествия · Творчество (walk-and-talk).

Avoid: total-gym-look, pastel-athleisure, цветной тренинг-wear, total-брендовый сет.

Style Profile: Boho (premium-этника)

Style Name: Boho (premium-этника)

Visual Code: свободный, природный, многослойный. Без костюмности и без цыганщины.

Base Wardrobe ♀: - Верх: длинная блуза с вышивкой, рубашка-туника, тонкий вязаный кардиган, топ с этнической вышивкой. - Низ: широкие льняные брюки, длинная юбка с воланами, миди-юбка из лёгкого хлопка. - Платья: длинное платье макси, миди с этнической вышивкой. - Обувь: замшевые сапоги, кожаные сандалии, mules с этническим декором. - Верхняя одежда: пончо, длинный кардиган, шерстяное пальто длинного силуэта. - Аксессуары: плетёная сумка, широкополая шляпа, кожаный ремень с пряжкой, длинные серьги.

Base Wardrobe ♂: - Верх: длинная льняная рубашка, рубашка-кафтан, лёгкая туника, тонкий вязаный кардиган. - Низ: широкие льняные брюки, прямые брюки с lite-карго-карманами. - Обувь: кожаные сандалии, замшевые мокасины, тёплые ботинки в межсезонье. - Верхняя одежда: длинный кардиган, шерстяной плед-пиджак, замшевая куртка. - Аксессуары: кожаный ремень с пряжкой, плетёная сумка-shopper, шейный платок.

Materials: лён, хлопок, шерсть, замша, плетение.

Color Logic: тёплые земляные — terracotta, sand, sage, rust, cream.

Social Signal: свобода, артистичность, природность.

Best Scene Fit: Путешествия · Очаг · Творчество · Социальность (камерный круг).

Avoid: бохо-дача (цыганщина), марафон-ярмарка, hippie-каприз, total-prints.

Base Wardrobe ♂: - Верх: графический long-sleeve, baggy-tee, jersey, ретро-футболка. - Низ: baggy jeans, low-rise джинсы, широкие карго. - Обувь: doc-boots, массивные кроссовки. - Верхняя одежда: милитари-jacket, кожаный bomber, пуховик-кроп. - Аксессуары: NY-cap, цепи, мини-сумка через плечо, очки овал / прямоугольник, массивный ремень.

Materials: деним, нейлон, металлик-ткань, плотный трикотаж, коттон-футер.

Social Signal: молодость, fashion-смелость, культурный код Tokyo / Seoul.

Best Scene Fit: Творчество · Социальность (молодой круг) · Путешествия (Tokyo / Seoul).

Avoid: cosplay-тотальный, рейв-костюм, «девочка-из-2003» (cringe-ностальгия), total-pink.

Materials: шёлк, тонкая шерсть (костюм), бархат (акцент), кожа.

Color Logic: чёрный, графит, nude / peach, красный акцент, золото. Без неона и без блеска.

Social Signal: масштаб, вкус-на-пике, сексуальность через сдержанность.

Best Scene Fit: Проявленность (выход в свет) · Отношения (ресторан, вечер) · Социальность (высокий круг).

Avoid: стразы-везде, bodycon-мини, Miami-клуб, голая-грудь-♂, неоновая подсветка.

Style Profile: Romantic / Feminine

Style Name: Romantic / Feminine

Visual Code: чувственная мягкость, drapery, утренняя естественность. Не свадебный, не cottagecore.

Base Wardrobe ♀: - Верх: лёгкая блузка с рукавом-фонариком, шёлковая блузка с воланом. - Платья: кремовое шёлковое slip-платье миди на бретелях; миди в gauze-ткани с открытыми плечами; длинное платье с тонкой драпировкой. - Обувь: босиком в сцене, мягкие балетки, мюли с лёгким каблуком. - Аксессуары: тонкие золотые цепочки, шёлковая лента-завязка в волосах, мини-сумка в кремовой коже.

Materials: шёлк, тонкий лён, gauze-ткань, замша, лёгкий трикотаж.

Color Logic: off-white, cream, blush, soft sand, тёплый янтарный подсвет. Без чёрного, без холодных тонов.

Social Signal: нежность, поэтика, внутренняя мягкость, тёплая женственность.

Best Scene Fit: Очаг · Отношения · Творчество (студия) · Образ-Я (внутренняя мягкость).

Avoid: свадебный наряд, cottagecore-дача, «молочный» total look, pastel-disney.

Использование в системе

Этот файл — рабочий вход для AI-движка генерации сцен Elyne. Каждый Style Profile подаётся в промпт-пайплайн вместе с:

  1. Сценарной зоной (одна из 10 сфер — Состояние / Ресурсы / Реализация / Партнёрство / Дом / Род / Творение / Развитие / Свобода / Окружение).
  2. Гендером пользователя — выбирает между блоком ♀ и блоком ♂ в Base Wardrobe.
  3. Возрастом пользователя.
  4. Q3 микроопросника (палитра ощущения / модус) — для совмещения с Color Logic стиля.
  5. Q4 / Q-prompt-form микроопросника (через что должно быть видно) — для совмещения с Best Scene Fit.

Антиформы (Avoid) обязательно прокидываются в негативный промпт.

Что НЕ зависит от гендера (и берётся из общего блока стиля): Materials, Color Logic, Social Signal, Best Scene Fit, Avoid.

Что зависит от гендера (берётся из соответствующего ♀ / ♂ блока): Base Wardrobe (верх / низ / платья / костюм / обувь / верхняя одежда / аксессуары).
