Юр-чек-лист для пресс-релиза Elyne API · 20.05.2026

1. Медицинский / психотерапевтический регламент

Красные флаги (никогда): - «помогает выйти из депрессии / тревоги / кризиса» - «терапевтический эффект», «психологическая помощь», «меняет психику» - «лечение», «восстановление», «реабилитация»

Риск: ФЗ-323 «Об охране здоровья» + ФЗ-38 ст.24 (реклама медуслуг). Без лицензии — административка, при систематике — уголовка ст.235 УК.

Зелёные альтернативы: - «инструмент самоисследования» - «визуализация желаемого образа жизни» - «помогает сформулировать и прочувствовать жизненные ориентиры» - «для людей, которые ищут следующий уровень себя»

Граница: говоришь о процессе (увидеть, почувствовать, сформулировать), не о результате (стать здоровым, вылечиться, восстановиться).

2. Персональные данные / фото (152-ФЗ)

В пресс-релизе достаточно одной фразы-якоря. Не разворачивать политику ПДн в PR-тексте, но обозначить факт обработки.

Формула:

«Elyne обрабатывает фотографии пользователей исключительно для генерации персонализированных сцен. Данные не передаются третьим лицам. Подробнее — в Политике конфиденциальности на [домен].»

Важно: к 20.05 должна быть живая страница с политикой ПДн. Если ссылки нет — она в пресс-релизе превращается в доказательство нарушения.

3. Manifestation-зона / ФЗ-38

ФЗ-38 ст.5 ч.3 — запрет недостоверной рекламы. Ст.5 ч.7 — запрет формирования искажённого образа объекта.

Красные флаги: - «увидите своё будущее» (претензия на предсказание) - «изменит вашу жизнь» (гарантия результата) - «позволяет притянуть желаемое» (псевдонаука)

Зелёные альтернативы: - «помогает создать детальный образ желаемой жизни» - «помогает найти направление» - «прочувствовать образ будущего»

Ключевой принцип: описываешь инструмент (что делает продукт технически), а не результат для пользователя (что произойдёт с его жизнью).

4. Юрлицо в пресс-релизе

Прямо в теле — необязательно. Достаточно в блоке «О компании» (About) в конце:

«Elyne разрабатывается [ИП Фамилия / ООО Название], [страна регистрации], 2026.»

Оферта и реквизиты — на сайте, не в PR. Если юрлицо не определено к 20.05 — «Elyne» как бренд без раскрытия. Риск: если принимаешь деньги — реквизиты оператора платежей должны быть в оферте на сайте.

5. Цитаты основателя

Запрещено: - «первое в мире» (нужно доказательство, иначе ФЗ-38 ст.5 ч.3) - «гарантирует результат X» - «единственное решение»

Разрешено: - «один из первых продуктов, который…» - «мы создали инструмент, который…» - субъективные формулировки («я верю», «наша цель — »)

6. ERID-маркировка

Органичный пресс-релиз (редакционный, без оплаты) — ERID не нужен.

ERID обязателен при: - оплаченном размещении в СМИ или у блогера - бартерной интеграции у инфлюенсера (даже без денег) - пресс-релизе, который блогер «публикует как свой текст»

Каждый канал распространения фиксируется отдельно. Если блогер берёт текст по бартеру — ERID через ОРД до публикации.
