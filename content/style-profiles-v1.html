Style Profiles · 10 стилей

Формат жёсткий, по шаблону. Все поля — короткие, конкретные, визуальные.

Style Profile: Old Money

Style Name: Old Money

Visual Code: сдержанный, статусный, без логотипов. Тихая роскошь, наследуемый вкус.

Base Wardrobe: - Верх: блейзер, поло, белая рубашка, кашемир-свитер. - Низ: юбка-миди, прямые брюки, светлые джинсы. - Обувь: лоферы, balmoral, балетки. - Верхняя одежда: пальто из шерсти, тренч. - Аксессуары: часы, шарф, кожаная сумка одной премиум-формы.

Materials: кашемир, твид, шёлк, тонкая шерсть, лён, премиум хлопок.

Color Logic: тёплые нейтральные — beige, cream, navy, camel, soft brown, off-white. Один акцент на образ.

Social Signal: статус без демонстрации, ухоженность, наследуемый вкус, спокойная сила.

Best Scene Fit: Достаток · Образ-Я · Отношения (ресторан, ужин) · Социальность (избранный круг).

Avoid: блестящие логотипы, trend-асимметрия, urban street, технологичные ткани, brand-overload.

Style Profile: Business Style

Style Name: Business Style

Visual Code: структурный, властный, современная деловая эстетика. Читаемая компетентность.

Base Wardrobe: - Верх: рубашка, блузка, тонкий джемпер. - Низ: брюки структурного кроя, юбка-карандаш. - Обувь: лоферы, derby, лодочки на низком каблуке. - Верхняя одежда: костюм слим / чуть-оверсайз, пальто-кокон. - Аксессуары: часы, ремень, минимальные украшения, портфель.

Materials: качественная шерсть, хлопок-поплин, шёлк (галстук/блузка), плотный трикотаж, кожа.

Color Logic: графит, чёрный, белый, navy. Бордо или глубокий зелёный как редкий акцент.

Social Signal: компетентность, сила, контроль, ясность.

Best Scene Fit: Проявленность · Достаток · Образование (лекция, конференция).

Avoid: широкий галстук 90-х, canvas-рюкзак, мятая рубашка, pastel-оттенки, двубортный «корпоратив-2000».

Style Profile: Classic

Style Name: Classic

Visual Code: вечная база, без субкультурного шума. То, что не устаревает.

Base Wardrobe: - Верх: белая рубашка, чёрный тонкий джемпер. - Низ: бежевые брюки, прямые джинсы, чёрное платье-футляр. - Обувь: балетки, лоферы, кожаные туфли на низком каблуке. - Верхняя одежда: тренч, шерстяное пальто. - Аксессуары: пояс, часы, лаконичная сумка.

Materials: хлопок, шерсть, кожа, плотный трикотаж, лён (межсезонье).

Color Logic: чёрный, белый, беж, navy в мягких сочетаниях. Принт — клетка, полоска, пейсли (редко).

Social Signal: вкус без претензии, надёжность, мягкая уверенность.

Best Scene Fit: Образ-Я · Семья (нейтральная база) · Социальность · Образование.

Avoid: subculture-маркеры, яркая мода, избыточный декор, total-trend-look.

Style Profile: Casual

Style Name: Casual (minimal-luxe)

Visual Code: повседневный, но материал говорит за себя. Не выглядит как усилие.

Base Wardrobe: - Верх: плотная белая футболка, кашемир-свитер, рубашка-оверсайз. - Низ: прямые джинсы, льняные брюки. - Обувь: lifestyle-кроссовки (не спорт), лоферы. - Верхняя одежда: кардиган, лёгкая куртка-shacket. - Аксессуары: одна сумка, часы.

Materials: кашемир, плотный хлопок, лён, замша, плотный трикотаж.

Color Logic: off-white, soft grey, warm beige, muted navy.

Social Signal: вкус, лёгкость, естественность, благополучие.

Best Scene Fit: Семья · Образ-Я · Творчество (home-studio) · Образование (коворкинг).

Avoid: спортзал-look, brand-overload, pastel-шум, fast-fashion-силуэт.

Style Profile: Streetwear

Style Name: Streetwear (premium)

Visual Code: уличный fashion-forward, не массовый hip-hop. Точный крой при оверсайзе.

Base Wardrobe: - Верх: премиум-худи, графический long-sleeve, оверсайз-рубашка. - Низ: прямые/широкие брюки, премиум-деним. - Обувь: массивные lifestyle-кроссовки (не беговые). - Верхняя одежда: анорак, оверсайз-куртка, бомбер. - Аксессуары: crossbody, beanie, кепка.

Materials: ripstop, нейлон, плотный premium-хлопок, деним, технический трикотаж.

Color Logic: графит, хаки, приглушённый вишнёвый, выцветший индиго. Бренд-акцент — редко и точечно.

Social Signal: креативность, fashion-литература, городская независимость.

Best Scene Fit: Творчество · Социальность · Путешествия (city break) · Проявленность (креативные индустрии).

Avoid: total-look бренда, sportswear, fast-fashion подделки, sneakerhead-нагромождение.

Style Profile: Sport Casual

Style Name: Sport Casual (function + luxury)

Visual Code: спорт-повседневный с одним luxury-якорем (часы / куртка / сумка).

Base Wardrobe: - Верх: худи, технический long-sleeve, тонкий бомбер. - Низ: технические брюки, jogger-крой. - Обувь: lifestyle-кроссовки одной премиум-модели. - Верхняя одежда: лёгкая shell-куртка, пуховик slim-fit. - Аксессуары: часы или сумка как luxury-якорь.

Materials: технический хлопок, флис, мембраны (Gore-Tex / shell), плотный трикотаж.

Color Logic: чёрный, графит, slate-blue, олива. Белый как контраст.

Social Signal: дисциплина, ритм, тихий статус.

Best Scene Fit: Образ-Я (утро, тренировка) · Путешествия · Творчество (walk-and-talk).

Avoid: total-gym-look, pastel-athleisure, цветной тренинг-wear, total-брендовый сет.

Style Profile: Boho

Style Name: Boho (premium-этника)

Visual Code: свободный, природный, многослойный. Без костюмности и без цыганщины.

Base Wardrobe: - Верх: длинная блуза, рубашка с вышивкой, тонкий вязаный кардиган. - Низ: широкие брюки, длинная юбка, длинное платье. - Обувь: замшевые сапоги, кожаные сандалии. - Верхняя одежда: пончо, длинный кардиган, шерстяное пальто. - Аксессуары: плетёная сумка, широкополая шляпа, кожаный ремень с пряжкой.

Materials: лён, хлопок, шерсть, замша, плетение.

Color Logic: тёплые земляные — terracotta, sand, sage, rust, cream.

Social Signal: свобода, артистичность, природность.

Best Scene Fit: Путешествия · Очаг · Творчество · Социальность (камерный круг).

Avoid: бохо-дача (цыганщина), марафон-ярмарка, hippie-каприз, total-prints.

Style Profile: Y2K

Style Name: Y2K (Japanese street-2000)

Visual Code: поп-эстетика 2000-х в современной подаче. Цитата, не косплей.

Base Wardrobe: - ♀: mini-юбка или low-rise джинсы, metallic-топ, baby-tee, кроп. - ♂: NY-cap, милитари-jacket, baggy jeans, массивный ремень. - Обувь: doc-boots, массивные кроссовки, платформы. - Верхняя одежда: кожаная куртка-bomber, пуховик-кроп, милитари-jacket. - Аксессуары: солнцезащитные очки овал/прямоугольник, мини-сумка, цепи.

Materials: деним, нейлон, металлик-ткань, плотный трикотаж, коттон-футер.

Color Logic: серебро, металлик, pastel-pop, чёрный как база.

Social Signal: молодость, fashion-смелость, культурный код Tokyo / Seoul.

Best Scene Fit: Творчество · Социальность (молодой круг) · Путешествия (Tokyo / Seoul).

Avoid: cosplay-тотальный, рейв-костюм, «девочка-из-2003» (cringe-ностальгия), total-pink.

Style Profile: Glam

Style Name: Glam (premium-sexy)

Visual Code: уверенность + скульптурная сексуальность. Вечерняя эстетика. Premium, не Vegas.

Base Wardrobe: - ♀: шёлковое slip-платье в пол на тонких бретелях, разрез до середины бедра. - ♂: графитовый slim-fit-костюм, шёлковая рубашка peach/nude расстёгнута на 3 пуговицы, карманный платок. - Обувь: ♀ strappy heels; ♂ derby, monk-strap. - Аксессуары: ♀ золотые украшения (серьги, цепочки), клатч; ♂ золотые часы, тонкий ремень.

Materials: шёлк, тонкая шерсть (костюм), бархат (акцент), кожа.

Color Logic: чёрный, графит, nude/peach, красный акцент, золото. Без неона и без блеска.

Social Signal: масштаб, вкус-на-пике, сексуальность через сдержанность.

Best Scene Fit: Проявленность (выход в свет) · Отношения (ресторан, вечер) · Социальность (высокий круг).

Avoid: стразы-везде, bodycon-мини, Miami-клуб, голая-грудь-♂, неоновая подсветка.

Style Profile: Romantic / Feminine

Style Name: Romantic / Feminine

Visual Code: чувственная мягкость, drapery, утренняя естественность. Не свадебный, не cottagecore.

Base Wardrobe: - ♀: кремовое шёлковое slip-платье миди на бретелях, лёгкая блузка с рукавом-фонариком. - ♂: кремовая льняная рубашка с закатанными рукавами, кремовые чиносы с отворотом. - Обувь: ♀ босиком в сцене или мягкие балетки; ♂ тканевые loafers, замшевые сандалии. - Аксессуары: тонкие золотые цепочки, шёлковая лента-завязка в волосах.

Materials: шёлк, тонкий лён, gauze-ткань, замша, лёгкий трикотаж.

Color Logic: off-white, cream, blush, soft sand, тёплый янтарный подсвет. Без чёрного, без холодных тонов.

Social Signal: нежность, поэтика, внутренняя мягкость, тёплая женственность.

Best Scene Fit: Очаг · Отношения · Творчество (студия) · Образ-Я (внутренняя мягкость).

Avoid: свадебный наряд, cottagecore-дача, «молочный» total look, pastel-disney.

Использование в системе

Этот файл — рабочий вход для AI-движка генерации сцен Elyne. Каждый Style Profile подаётся в промпт-пайплайн вместе с:

  1. Сценарной зоной (одна из 10 сфер — Состояние / Ресурсы / Реализация / Партнёрство / Дом / Род / Творение / Развитие / Свобода / Окружение).
  2. Гендером и возрастом пользователя.
  3. Q3 палитрой ощущения (из микроопросника — для совмещения с Color Logic стиля).
  4. Q4 prompt-form (через что должно быть видно — для совмещения с Best Scene Fit).

Антиформы (Avoid) обязательно прокидываются в негативный промпт.
