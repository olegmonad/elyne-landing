08 · photo-upload
Загрузка фото профиля
Mandatory → identity anchor (главный инпут генерации)
Фото профиля
Загрузите от 1 до 10 фотографий. Выберите главное фото, нажав на звёздочку.
  • Чекбокс: «Я разрешаю использовать мои фотографии для генерации»
  • Без чекбокса → кнопка Далее заблокирована
Нет объяснения ЗАЧЕМ нужно фото. По данным UX-аудита ~20% пользователей дропаются именно здесь. Рекомендация: добавить 1 строку «Мы используем фото только для создания твоих сцен».
